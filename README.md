##### Prerequisites

- Node.js 16.x or higher
- npm or yarn

##### Installation

1. Clone the repository:

```bash
git clone https://github.com/ethannhzhouu/ethan-portfolio.git
cd ethan-portfolio

2. Install 
npm install

3. Run the development server:
npm run dev
```
4. Open [http://localhost:3000](http://localhost:3000).
